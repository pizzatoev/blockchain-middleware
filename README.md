# Proyecto Blockchain BaaS
Bienvenido al repositorio central.

## Instrucciones para el equipo (Solo la primera vez):
1. Clona este repositorio.
2. Abre Docker Desktop.
3. Abre tu terminal y entra a: `cd red-hyperledger/test-network`
4. Enciende la red con: `./network.sh up`
*(Nota: la primera vez tardará porque descargará las imágenes de Docker)*.

Para apagar la red y limpiar: `./network.sh down`
